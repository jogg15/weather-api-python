### Clone and Enter:
```git clone https://github.com/jogg15/weather-api-python.git```<br>
```cd weather-api-python```

### Install Dependencies
```python -m pip install --upgrade pip```<br>
```python -m pip install -r requirements.txt```
#### In case of MSVC error: <br>```python -m pip install --only-binary :all: aiohttp```

### Configuration
Create .env file inside the project:<br>
- insert ```OPENWEATHER_API_KEY=your_api_key_here```

### Execution
```python main.py```
